Q1. Name two types of Java programs. Give the difference between them.
Ans. Java can be used to create two types of programs: applications and applets. An application is a program that runs on your computer, under the operating system of that computer. Java programs that work on Web pages are called applets. To use an applet, you need a Java-compatible Web browser, which will interpret the bytecodes for you.


Q2. What do you mean by bytecode? How can a Java code run on all platforms?
Ans. When a Java program is compiled, the Java compiler called javac.exe (an executable file) creates a file with .class extension called bytecode. The Java bytecode is 100% hardware/ operating system/ platform independent, intermediate binary code. Thus, the Java bytecode code is completely portable and secure and can be run on all types of hardware configurations or environments with different operating systems. The Java bytecode is run or executed by Java Interpreter called java.exe (an executable file), also called Java run-time system or Java Virtual Machine (JVM).


Q3. What is Java run-time system or Java Virtual Machine (JVM)?
Ans. The Java bytecode is run or executed by Java Interpreter called java.exe (an executable file), also called Java run-time system or Java Virtual Machine (JVM).


Q4. What is an Integrated Development Environment (IDE)? Name some IDE for Java Programming Language.
Ans. An integrated development environment (IDE) is a software application that provides facilities to computer programmers for software development. An IDE creates a work place, where you type the program or source code and do editing and debugging. It provides links to compile and execute a program. Some IDE for Java Programming Language are NetBeans, Eclipse, TextPad, JCreator, BlueJ etc.


Q5. What is a token?
Ans. Java program is a collection of identifiers, keywords, literals, operators, comments, separators and white spaces. These individual units of a Java program are called tokens.


Q6. What is an identifier?
Ans. Identifier is used for class name, method name, variable name and reference variable name.


Q7. What are rules of writing identifier?
Ans. An identifier may be any descriptive sequence of uppercase and lowercase letters, numbers, or the underscore and dollar sign characters. They must not begin with a number. Keywords or reserve words of Java language cannot be used as an identifier. Java is case-sensitive.


Q8. What are keywords or reserve words?
Ans. Words that have a special meaning to a computer language compiler or interpreter are called keywords or reserve words.


Q9. There are two types of memory locations – variable and reference variable. State the difference between them.
Ans. Memory location in which data is stored is called a variable. On the other hand memory location in which address of some other memory location or object is stored is called a reference variable.


Q10. What do you mean by block or compound statement?
Ans. A block or compound statement is any number of simple Java statements that are surrounded by a pair of braces. Blocks define the scope of variables.


Q11. All variables have a scope. Explain.
Ans. Scope defines the visibility and lifetime of the variable. Scope is the program area where the variable can be accessed.


Q12. How can a variable be declared as a constant?
Ans. A variable can be declared as final to prevent its content from being modified. Thus, a final variable is essentially a constant.
final double pi=3.14;


Q13. What are comments and why are they used? Give any two types of comments in Java.
Ans. Comments are remarks, which may be entered into a program. The contents of a comment are ignored by the compiler. Comment describes the operation of the program and may remind you of the name of source file. A multi-line comment begins with /* and ends with */.

A single line comment begins with a // and ends at the end of the line.

Q23. What is the difference between rvalue and lvalue?
Ans. rvalue (read-value) is the data stored in a memory location while lvalue (location-value) is the actual address where the data is stored. A variable of primitive data type stores rvalue while a variable of reference type stores lvalue.

Q25. What are literals? Name the different types of literals.
Ans. A literal is the source code representation of a fixed value represented directly in the code without requiring computation. Literals may be integer literals, floating point literals, character literals, string literals, boolean literals and null literal. Examples of literals may be 189, 36.45, ‘A’, “Simran”, true etc.


Q26. What are integer literals? How are they stored in memory?
Ans. Integer literals are stored as int and thus occupy 32 bits space in memory. Decimal (base 10), octal (base 8), hexadecimal (base 16) and binary (base 2) numbers can be used as integer literals. The range of an octal digit is 0 to 7, the range of hexadecimal digit is 0 to 15, where A through F (or a through f) is substituted for 10 through 15 and the binary digits consists of 0 and 1. A leading 0 denotes octal values such as 0126, 045 etc., leading 0x denotes hexadecimal values such as 0xBAD, 0xDAD007, 0x918CAFE27 etc. and the leading 0b denotes binary values such as 0b101101, 0b10010 etc. The integer literals are stored as long if it ends with the letter L or l, and in this case, it occupies 64 bit space in memory.


Q27. How can you explicitly specify an integer literal as long?
Ans. An integer literal can be explicitly specified as long by appending L or l to the literal.
e.g. 153L or 90l.


Q28. How can you explicitly specify an integer literal as double?

Q29. What are floating-point literals? How are they stored in memory?
Ans. Floating-point literals are stored as data type double and thus occupy 64 bits space in memory. The floating-point literals are stored as float if it ends with the letter F or f and in this case, they occupy 32 bit space in memory. They represent decimal values with a fractional component. They may be expressed in either standard notation such as 0.98145, 8.19054 etc. or scientific notation such as 2346318E-9, 3.672e+54 etc. or 3.15F or 6.3f.


Q30. How can you explicitly specify a floating-point literal or an integer literal as float?

Q39. What do you mean by type conversion? Give the difference between implicit and explicit type conversion.
Ans. Converting one data type to another is called type conversion. In an assignment statement, if source type is smaller than the destination type, implicit conversion (also called automatic type conversion, coercion, promotion or widening conversion) takes place. For example
int y = 9;
float z = y;
In an assignment statement, when the source type is larger than the destination type, explicit type conversion (also called casting or narrowing conversion) takes place. For example in the code fragment written below, we are explicitly making the value narrower so that it will fit into the target type.
double m = 72.9;
int n = (int)m;
In the above case, the fractional component is lost or truncated. Usually during explicit type conversions, loss of information is possible.

Q48. What is a ternary or conditional operator and how does it work? Give its general form.
Ans. Ternary operator ?: along with its operands forms an expression which evaluates to a single value namely data or object reference depending on the conditional expression it evaluates. The value evaluated by such expression may be assigned to a variable or a reference variable, or used in other expression, or displayed, or returned to the calling method as the case may be. It is called ternary operator because it works or operates on three operands. A ternary operator can replace certain types of if-else statements.
The ? has this generalized form:
expression1 ? expression2 : expression3
Here expression1 can be any expression that evaluates to a boolean value. If expression1 is true, expression2 is evaluated; otherwise, expression3 is evaluated. For example the following code fragment stores in ‘g’ the greater of the two numbers in x and y respectively.
int x=9,y=27,g;
g=x>y?x:y;


Q49. What is purpose of assignment operator? Give its general form. How does assignment operator allow creation of chain of assignments?
Ans. The assignment operator is used to store data in a variable, or address of a memory location or object in a reference variable. The assignment operator is the single equal sign, =. It has this generalized form:
var = expression;
Here, the type of var must be compatible with the type of the expression.
The assignment operator allows creation of chain of assignments. For example, consider this code fragment:
int a, b, c;
a = b = c = 99;
This fragment sets the variables a, b and c to 99 using a single statement.

Associativity decides the order in which, the expression having operators of the same group, is to be evaluated, whether left to right or right to left.

Q54. Explain the if-else if ladder. Give the general form of if-else if ladder.
Ans. The conditional expressions of each if statement is tested from top to bottom. As soon as one of the condition controlling if statement is true, the statement or compound statement associated with that if statement is executed, and the rest of the ladder is bypassed. If the entire conditional tests fail, then the last else statement is performed. The final else is optional. The general form of the if-else if ladder is:
if(condition) statement or compound statement
else if(condition) statement or compound statement
else if(condition) statement or compound statement
………
else statement or compound statement


Q55. Explain the switch statement. Give the general form of a switch statement. What is the purpose of break and default in the switch statement?
Ans. The switch statement is a multi-way branch statement. It provides an easy way to transfer the control to different parts of the code based on the value of an expression. It is a better alternative of a large series of if-else if statements. The general form of a switch statement is:
switch(expression)
{
case value1:
//statement sequence
break;
case value2:
//statement sequence
break;
………
case valueN:
//statement sequence
break;
default:
//default statement sequence
}

In the switch statement, the value of the expression is compared with each literal value in the case statements. If match is found, the code sequence following that case statement is executed. If none of the literals match with the value of the expression, then the default statement is executed. The break statement inside the switch bypasses the remaining code in the body of the switch and the program control resumes at the next statement following the entire switch statement.


Q56. What do you mean by fall through?
Ans. The break statement is optional in a switch statement. If a break statement is omitted, execution will continue into the next case. If all the break statements in the switch statement are omitted, all the cases in the switch will be executed. This is called fall through.


Q57. Explain for loop. Give the general form of for loop.
Ans. When the loop starts, the initialization portion of the loop is executed, that sets the initial value of the loop control variable. The initialization expression is executed only once. Next, condition expression is evaluated. The condition expression usually tests the loop control variable against a target value. If this conditional expression is true, then the statements in the body of the loop are executed, if it is false, the loop terminates. Next, the update expression or iteration portion of the loop is executed. An update expression changes (usually increments or decrements) the value of the loop control variable so to make the conditional expression, which is controlling the loop, false at a later stage and forcing the loop to terminate. Again, conditional expression is evaluated and so on. This process repeats until the condition is false.
Here is the general form of for statement:
for(initialization expression; conditional expression/ test expression/ termination expression; iteration/ update expression)
{
//body of loop
}


Q58. What do mean by body of a loop? What is update expression in the for loop?
Ans. The body of the loop is a statement or a set of statements, which are repeatedly executed, until the conditional expression controlling the loop is true.
In the for loop if this conditional expression is true, then the statements in the body of the loop are executed, if it is false, the loop terminates. Next, the update expression or iteration portion of the loop is executed. An update expression changes (usually increments or decrements) the value of the loop control variable so to make the condition expression, which is controlling the loop, false at a later stage and forcing the loop to terminate.


Q59. Explain the while loop. Give the general form of a while loop.
Ans. The while loop repeats, a statement, or block or the body of the loop, while its conditional expression is true. Its general form is:
while(condition)
{
//body of loop
}
When condition becomes false, control passes to the next line of code immediately following the loop. The curly braces are unnecessary if only a single statement is being repeated.


Q60. Explain the do-while loop. Give the general form of a do-while loop.
Ans. If the conditional expression controlling a while loop is initially false, then the body of the loop will not be executed at all. However, in a do-while loop the body of the loop executes at least once, even if the conditional expression is false to begin with, because its’ conditional expression is evaluated at the bottom of the loop. Its general form is:
do
{
//body of loop
}
while(condition);
In all iteration statements, condition must be a boolean expression.


Q61. Give the difference between while and do-while loop.
Ans. If the conditional expression controlling a while loop is initially false, then the body of the loop will not be executed at all. However, in a do-while loop the body of the loop executes at least once, even if the conditional expression is false to begin with because its conditional expression is evaluated at the bottom of the loop.


Q62. Which loop is often used when you know in advance the number of times a loop will iterate or repeat itself?
Ans. The for loop is often used when we know in advance, the number of times, a loop will iterate or repeat itself.


Q63. Which loops are often used when you do not know the number of times a loop will iterate or repeat itself?
Ans. The while and do-while loops are often used when we do not know, the number of times a loop will iterate or repeat itself.


Q64. Which loop is called entry-controlled loop and why?
Ans. The for loop and while loop are entry-controlled loops because the conditional expression controlling the loop is evaluated before entering the body of the loop.


Q65. Which loop is called exit-controlled loop and why?
Ans. The do-while loop is exit-controlled loop because it evaluates the conditional expression controlling the loop at exit point after entering the body of the loop at least once.


Q66. What is the difference between break and continue statement?
Ans. When a break statement is encountered inside the loop or the switch, the remaining code in the body of the loop or the switch as the case may be, is bypassed. The program control resumes at the next statement following the body of the loop or the entire switch statement. When a continue statement is encountered inside a loop, any remaining code in the body of the loop is bypassed and the loop continues running.


Miscellaneous Theoretical Questions and Answers


Q1. Some of the primitive data types of Java are int, long, char, float, double etc. What do these primitive data types decide?
Ans. The data type of a variable decides firstly the amount of space occupied by the data in the memory (in bits or bytes). Secondly, it decides the type of data, which can be stored in that variable whether integer, real number, character or truth-value true or false. Lastly, it defines in most cases, a range of numbers or a set of characters as the case may be, from which a number or character can be stored in that variable.

Ans. Separators are symbols used to indicate where groups of code are divided and arranged. The nine Unicode characters ( ) { } [ ] ; , . are referred to as separators or punctuators.
parenthesis ( ) used to enclose parameters in method definition and invocation, also for defining precedence in expressions, containing expressions for flow control, and surrounding cast types

Q5. Give two differences between switch and if statement?
Ans. The hidden conditional expression in switch statement can only test equality, whereas conditional expression in if statement can evaluate any type of relational operation. In switch statement, the data type of the expression, which is compared with the literal value, cannot be long, float, double or boolean, but conditional expression in if statement can compare all primitive data types and literals. Lastly the hidden conditional expression in switch statement cannot handle range of data, whereas conditional expression in if statement can express ranges.

An else matches with the nearest, previous, unmatched if that is not in a block.

Q12. What is the difference between the statement c=c+1; and its shorthand form ++c;?
Ans. The expression ++c does not always reflect the same meaning as expression c=c+1, as explained in many books of Java Programming. Both the expressions mean that the content of variable c is incremented by 1. However, few programmers know the technical difference between them. Expression c=c+1 takes type conversion into consideration. On the contrary, it is not the same case in the expression ++c. Let us see the output of two code fragments given below.

for(byte p=1;p<=5;++p)System.out.print(p+” “);
The output of the code fragment given above is 1 2 3 4 5.

In the code fragment given below expression ++p is replaced by p=p+1.
for(byte p=1;p<=5; p=p+1)System.out.print(p+” “);
This code generates an error in expression p=p+1.

Reason: In expression p=p+1 data type of variable p is byte and 1 is a integer literal stored as data type int in memory. When arithmetic is performed between data of type byte and type int in expression p+1 , the result is int i.e. higher data type. Now, data of higher type cannot be stored as lower data type.

Thus, in expression p=p+1, the data type of the result of p+1 is to be converted to data type byte as given below.
p=(byte)(p+1);

After correction the second code fragment is
for(byte p=1;p<=5; p=(byte)(p+1))System.out.print(p+” “);
and the output is now same as the first code fragment.
